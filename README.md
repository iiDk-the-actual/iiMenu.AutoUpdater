# iiMenu.AutoUpdater
An automatically updating version of [ii's Stupid Menu](https://github.com/iiDk-the-actual/iis.Stupid.Menu) for Gorilla Tag

---

# Installation

- Download the release from [here](https://github.com/iiDk-the-actual/iiMenu.AutoUpdater/releases/latest)
- Drag the file `iiMenu.AutoUpdater.dll` to your plugins folder
- Start Gorilla Tag if needed
